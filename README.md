# SQL Practice Exercises

This repository contains a series of SQL exercises designed to strengthen fundamental SQL skills, including working with aggregate functions, grouping, and conditional logic using the `CASE` statement.

## Exercises

### Exercise 1 - SQL Fundamentals
- Focuses on basic SQL queries.
- Covers `SELECT`, `FROM`, `WHERE`, and simple filtering.
- Helps build a strong foundation for querying databases.

### Exercise 2 - Aggregate Functions & Grouping
- Introduces aggregate functions: `COUNT()`, `SUM()`, `AVG()`, `MIN()`, `MAX()`.
- Covers `GROUP BY` to summarize data by categories.
- Includes `HAVING` to filter grouped results.

### Exercise 3 - SQL Case Statement
- Teaches conditional logic using the `CASE` statement.
- Enables dynamic transformations of data in queries.
- Can be combined with aggregate functions and grouping for advanced analysis.
